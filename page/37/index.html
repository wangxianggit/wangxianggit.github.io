Python __slot__变量

  • ​ Python可以在定义类中添加特殊变量 __slot__ 来限制该类可以添加的属性，例：

    • ~~~
      class Student(object):

       __slots__ = ('name', 'age') # 定义允许绑定的属性名称

    • 由于score没有被放到__slots__中，所以不能绑定score属性，试图绑定score将得到AttributeError的错误。

    • __slot__定义的属性仅对当前的类起作用，对继承的子类不起作用。

python3.6自动安装脚本

  • ​ linux版本要求：CentOS，Scientific Linux

  • 自动安装脚本

    • #!/bin/bash
yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel
mkdir -p /usr/local/python3
cd /mnt
wget https://www.python.org/ftp/python/3.6.1/Python-3.6.1.tgz
tar -zxvf Python-3.6.1.tgz
cd Python-3.6.1
./configure --prefix=/usr/local/python3
make && make install
ln -s /usr/local/python3/bin/python3 /usr/bin/python3
echo "PATH=$PATH:/usr/local/python3/bin"  >>~/.bashrc
echo "export PATH " >>~/.bashrc
source ~/.bashrc
python3 -V

    • 环境变量可以设置在~/.bashrc或/etc/profile

PyPI使用国内源

  • ​ 国内主要的pip源有：

    • 阿里云 http://mirrors.aliyun.com/pypi/simple/
    • 中国科技大学 https://pypi.mirrors.ustc.edu.cn/simple/
    • 豆瓣(douban) http://pypi.douban.com/simple/
    • 清华大学 https://pypi.tuna.tsinghua.edu.cn/simple/
    • 中国科学技术大学 http://pypi.mirrors.ustc.edu.cn/simple/

  • 使用方法：最后加参数-i和指定的url，如：

    • pip install web.py -i http://pypi.douban.com/simple
或避免报错
pip install web.py -i http://pypi.douban.com/simple --trusted-host pypi.douban.com

U盘安装windows和CentOS7双系统

  • ​ 先安装windows后安装CentOS7

  • 进入安装界面后不能直接选择“Install CentOS 7”或“Test this media & istall CentOS 7”，安装前需要指定U盘的位置

  • 安装界面上按下Tab键，屏幕下方出现以下字符串：

    • ~~~

      Vmlinuz initrd=initrd.img inst.stage2=hd:LABEL=CentOS\x207\x20x86_64 4d.live.check quiet_

    • 看到界面上出现U盘的盘符或位置，接着重启电脑后重复上面操作将安装位置指定,后回车就可以进行安装

    • Vmlinuz initrd=initrd.img inst.stage2=hd:/dev/sdb4
