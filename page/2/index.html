Python使用cprint模块打印带颜色字符

  • cprint功能：在终端打印带有颜色的字符串

  • 安装：pip install cprint或下载安whl安装包地址https://pypi.org/project/cprint/

  • 使用示例：

    • from cprint import *

cprint(str) # WHITE
cprint.ok(str) # BLUE
cprint.info(str) # GREEN
cprint.warn(str) # YELLOW
cprint.err(str, interrupt=False) # BROWN
cprint.fatal(str, interrupt=False) # RED

Hbase启动集群有些节点没有HRegionServer

  • 问题：启动hbase集群，三个节点都有HMaster，其中一个节点没有HRegionServer

  • 查看日志出现时间不同步警告：

    • 2021-03-24 16:29:06,870 WARN  [RpcServer.default.FPBQ.Fifo.handler=27,queue=0,port=16000] master.ServerManager: Server hadoop1,16020,1616545792628 has been rejected; Reported time is too far out of sync with master.  Time difference of 28750450ms > max allowed of 30000ms

  • 使用date命令查看三个节点的时间，其中有一个节点时间不同。

  • 输入命令：ntpdate -u ntp.api.bz使时间同步并添加开机启动即可。
