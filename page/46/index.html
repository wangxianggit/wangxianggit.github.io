htop常用快捷键

  • htop常用快捷键

    • F1,h,?:查看htop使用说明，
F2,s  :设置选项
F3,/  :搜索进程
F4,\  :过滤器，输入关键字搜索
F5,t  :显示属性结构
F6,<,>:选择排序方式
F7, [,:减少进程的优先级(nice)
F8，] :增加进程的优先级(nice)
F9，k :杀掉选中的进程
F10，q:退出htop
u:显示所有用户，并可以选中某一特定用户的进程
U：取消标记所有的进程

ifstream重复读取文本文件

  • ifstream流对象关闭后不能改变流对象的内部状态。如果遇到读写失败，对象的状态将一直保持错误模式。

  • 使用clear操作可以重新恢复流对象的内部状态。

  • 若需要使用流对象重复读取某个文件，必须在重复读取文件之前调用clear函数清除流对象之前保留的状态。

  •   long bt, dt0, dt;
  int fnp, inp;
  string ss;
   ifstream infile(ipfn, ios::in);//ipfn:file name
        bool t=false;
        while(getline(infile,ss)){
          istringstream iss(ss);
            iss>>bt>>dt0>>dt>>fnp>>inp;
        }
        infile.close();
        infile.clear(ios::goodbit);
