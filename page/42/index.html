Python定制类

  • python中 __xxx__ 形式的变量或函数名在程序上有特殊用途。

  • __slot__用于指定类可以添加的属性。

  • __len__()用于返回类成员变量的长度。

  • __str__()自定义方式返回类对象的名称，例：

    • ~~~
      class Student(object):

       def __init__(self, name):
     self.name = name
 def __str__(self):
     return 'Student object (name: %s)' % self.name

      print(Student(‘Tom’))
      Student object (name: Tom)

    • 这里输入对象名称后python调用函数__repr__，输出实例名称。

  • 另外python中还有其他定制类如__iter__ , __getitem__等等。

Python特殊函数__len__

  • ​ Python特殊函数__len()__使函数len()能够作用于类中的变量，例：

    • class Namespace(object):

    def __init__(self, *args):
        self.names = args

    def __len__(self):
        return len(self.names)

s = Namespace('jia', 'tim')
print( len(s) )

Python __slot__变量

  • ​ Python可以在定义类中添加特殊变量 __slot__ 来限制该类可以添加的属性，例：

    • ~~~
      class Student(object):

       __slots__ = ('name', 'age') # 定义允许绑定的属性名称

    • 由于score没有被放到__slots__中，所以不能绑定score属性，试图绑定score将得到AttributeError的错误。

    • __slot__定义的属性仅对当前的类起作用，对继承的子类不起作用。

python3.6自动安装脚本

  • ​ linux版本要求：CentOS，Scientific Linux

  • 自动安装脚本

    • #!/bin/bash
yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel
mkdir -p /usr/local/python3
cd /mnt
wget https://www.python.org/ftp/python/3.6.1/Python-3.6.1.tgz
tar -zxvf Python-3.6.1.tgz
cd Python-3.6.1
./configure --prefix=/usr/local/python3
make && make install
ln -s /usr/local/python3/bin/python3 /usr/bin/python3
echo "PATH=$PATH:/usr/local/python3/bin"  >>~/.bashrc
echo "export PATH " >>~/.bashrc
source ~/.bashrc
python3 -V

    • 环境变量可以设置在~/.bashrc或/etc/profile
