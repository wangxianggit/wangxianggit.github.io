Python关键字assert

  • ​ 官方解释( https://docs.python.org/3/reference/simple_stmts.html#assert )：”Assert statements are a convenient way to insert debugging assertions into a program”

  • 一般用法：assert + 条件，如：assert 1==2

  • 输出结果：条件通过无输出，条件判断错误抛出异常

  • 例如：

    • >>>assert n<2
Traceback (most recent call last):
  File "<pyshell#3>", line 1, in <module>
    assert False
AssertionError
>>>assert 2>1
>>>assert 3<2
Traceback (most recent call last):
  File "<pyshell#3>", line 1, in <module>
    assert False
AssertionError

Python特殊函数__getitem__切片功能

  • ​ __getitem__函数传入参数可以是int也可以是切片对象

  • 若传入的是切片对象，则函数相应输出切片对象

  • 以费波拉契函数为例分别判断函数输入int和slice对象时的情形

    • ~~~
      class Fib(object):
      def __getitem__(self, n):
    if isinstance(n, int):
        a, b = 1, 1
        for x in range(n):
            a, b = b, a + b
        return a
    if isinstance(n, slice):
        start = n.start
        stop = n.stop
        a, b = 1, 1
        L = []
        for x in range(stop):
            if x >= start:
                L.append(a)
            a, b = b, a + b
        return L

  • 与之对应的是__setitem__()方法，把对象视作list或dict来对集合赋值。

  • __delitem__()方法，用于删除某个元素。

Python特殊函数__getitem__()

  • ​ 对对象按照下标取出元素，使用特殊函数__getitem__()

  • 以斐波那契数列作为对象，实现用下标得到相应类的实例：

    • class Fab(object):
    def __init__(self):
           self.a, self.b = 1, 1

    def __getitem__(self, n):
        for x in range(n):
            self.a, self.b = self.b, self.a + self.b
        return self.a

Python特殊函数__iter__()

  • ​ 循环调用一个类需要使用特殊函数方法__iter__(), 改方法返回一个迭代对象。

  • 在一个循环中会不断调用迭代对象的next()方法得到循环的下一个值。

  • 循环终止于抛出异常StopIteration()类的出现。

  • StopIteration是一个类，抛出异常时，会自动有一个中间环节，就是生成StopIteration的一个对象。

      raise StopIteration()

Python定制类

  • python中 __xxx__ 形式的变量或函数名在程序上有特殊用途。

  • __slot__用于指定类可以添加的属性。

  • __len__()用于返回类成员变量的长度。

  • __str__()自定义方式返回类对象的名称，例：

    • ~~~
      class Student(object):

       def __init__(self, name):
     self.name = name
 def __str__(self):
     return 'Student object (name: %s)' % self.name

      print(Student(‘Tom’))
      Student object (name: Tom)

    • 这里输入对象名称后python调用函数__repr__，输出实例名称。

  • 另外python中还有其他定制类如__iter__ , __getitem__等等。
