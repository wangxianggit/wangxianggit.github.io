AD原理图放置元件

放置元件技巧

当拖拽元件时，在元件随光标处于浮动状态时：

  • 按空格键，以90度增量逆时针旋转元件。
  • 按X键沿X轴翻转；按Y键沿Y轴翻转。
  • 按Tab键，可以在放置元件前修改对象的属性，输入值成为默认值，标识符自动递增。
  • 放置元件中，若到了原理图设计窗口的边界，软件自动移动页面。
  • 单击鼠标右键并拖拽，可以上下左右移动设计图纸。
  • 按下Ctrl+PgDn再次显示整个图纸。
  • 在库或Vault中找到元件并选择Place，该元件自动添加到原理图中。
  • 放置器件结束后，按Esc退出元件的放置过程。

改变元件位置技巧

  • 放置元件后重新调整位置可以选中该元件，然后按住鼠标左键并移动鼠标，则移动到指定位置。
  • 任何时候按快捷键G，修改Grid设置。用小的Grid设置，可以更精确定位元件。
  • 在原理图中，任意位置鼠标右键选择Option->Document Option->Units选择英制或mm单位。

AD布线中添加过孔和板层

AD布线中添加过孔和板层

  • 在AD软件交互布线中可以添加过孔，过孔不能再产生冲突的位置添加。
  • 过孔属性设计规则在PCB Rules and Constraints Editors对话框中的Routing via Style添加。

添加过孔并切换板层

  • 在布线过程中按数字键盘的“*”或“+”键添加一个过孔并切换到下一个信号层。
  • 按“-”键添加一个过孔并切换到上一个信号层。

添加过孔不切换板层

  • 按“2”键添加一个过孔，但是仍然保持在当前布线层。
  • 单击确定过孔位置。

添加扇出过孔

  • 按数字键盘的“/”键为当前走线添加过孔。
  • 用这种方法添加过孔后将返回原交互式布线模式，可以立刻进行下一处网络布线。
  • 这个功能在需要放置大量过孔时可以节省时间。

布线中的板层切换

  • 在多层板上的焊盘或过孔布线时，可以通过快捷键L把当前线路切换到另一个信号层中。
  • 若需要切换板层需要先添加过孔才能实现。

PCB板的单层显示

  • 在PCB显示中按快捷键Shift+S可以实现单层显示。
  • 选择哪一层的标签，就显示哪一层。
  • 在单层显示中按快捷键Shift+S可以返回多层显示模式。

原子核结合能和质量过剩

  • ​ 原子核的结合能公式：
    • 质量亏损： $\Delta M=[Zm_p+Nm_n]-M(A,Z)$
    • $m_n=1.008665m_u, m_p=1.007825 m_u$
    • $E_b=\Delta Mc^2=[(1.008665N + 1.007825 Z)m_u- M(A,Z)]\times931.478(MeV)$
    • $E_b$是正值
  • 原子核剩余质量：
    • 剩余质量： $\Delta M(A,Z)=(M(A,Z)-Am_u)\times 931.478+a_{el}Z^{2.39}(MeV)$
    • $a_{el}=1.433\times 10^{-5}(MeV)$
  • 原子核结合能和剩余质量关系推导：
    • $\Delta M(A,Z)=[(1.008665N + 1.007825 Z-A)\times 931.478 - E_b] +a_{el}Z^{2.39} (MeV)$

matplotlib动态画图

plt.ion() && plt.ioff()

  • matplotlib可视化库显示模式默认是阻塞模式，即执行图形显示后程序就会停止。
  • 若要继续执行程序，就要关闭图片。
  • 而plt.ion()这个函数使matplotlib显示模式转为交互模式。
  • 使用ion()命令可以开启了交互模式。
  • 在交互模式下plt.plot(x)或plt.imshow(x)是直接出图像，不需要plt.show()
  • 交互模式下，没有使用ioff()关闭的话，则图像会一闪而过，并不会常留。

硬盘接口比较

三种常见硬盘接口比较​

  • 常见硬盘分类有：IDE、SCSI、SATA，这三种表示不同接口标准。
  • 硬盘的通用标准是IDE（Integrated Device Electronics:电子集成驱动器的缩写），IDE经过不断的发展，除了开始的几个版本外，后续的几个版本都叫Ultra DMA，其中最快的是Ultra DMA133，速度达到了133M/s。
  • IDE硬盘通常也成为ATA硬盘，但严格来说IDE的范围比ATA更大，任何电子集成驱动器都属于IDE，甚至包括SCSI（Small Computer System Interface）。
  • 直到SATA的出现将ATA和IDE区分开来，SATA即Serial-ATA（串行ATA），而IDE则属于Parallel-ATA(并行ATA)。
  • 即廉价便宜的是IDE，新兴快速的是SATA，稳定高价的是SCSI。

IDE(Intergrated Device Electronics)

  • IDE一般是有扁平电缆连接的，一个扁平电缆可以连接2个IDE硬盘，而一个计算机一般提供2个扁平电缆，所以一般允许4块IDE硬盘。
  • 优点：成熟，廉价，稳定。一般老式的电脑中装的都是这种硬盘。

SATA(Serial-ATA)

  • SATA是串行ATA，是近些年的新兴技术，数据校验更为完整，传输速度也更快，快到什么程度呢？SATA1.0的标准已经达到150M/s了，至于后续的2.0和3.0，则是可以达到300M/S和600M/S的。
  • 总之SATA是一项新技术，在各个方面都比IDE更加出色，所以总的来讲，IDE迟早会被SATA完全取代的，只是时间问题而已。

SCSI(Small Computer System Interface:小型计算机系统接口)

  • SCSI的出现主要是由于原有IDE接口的硬盘转速比较低，传输效率比较慢。
  • 由于SCSI硬盘独立于系统总线工作，所以它的最大优点就在于系统的占用率低，当然它还有一些优点，转速高，传输速率快，更稳定。但也有一些缺点，价格高，安装没有IDE方便。
  • SCSI硬盘一般来讲只有服务器才用到的，在个人电脑上我们很少见到SCSI硬盘的存在。
