ssh启动错误：no hostkeys available— exiting

  • ssh服务启动不了，查看出现错误原因：

    • /usr/sbin/sshd -T
    • 若输出错误存在问题：no hostkeys available— exiting，则错误原因是ssh密钥出现问题。

  • root权限下，重新生成密钥：

    • ssh-keygen -t dsa -f /etc/ssh/ssh_host_dsa_key
    • ssh-keygen -t rsa -f /etc/ssh/ssh_host_rsa_key

  • 修改密钥权限：

    • chmod 600 /etc/ssh/ssh_host_dsa_key
    • chmod 600 /etc/ssh/ssh_host_rsa_key

  • 重启ssh服务即可。

cmake打印乘法表

cmake打印乘法表

  • CMakeLists.txt 文件代码：

    • cmake_minimum_required(VERSION 2.8.11) 
project(test) 
foreach(i RANGE 1 9) 
    set(exp "") 
    foreach(j RANGE 1 ${i}) 
        math(EXPR value "${j} * ${i}") 
        string(APPEND exp "${j}" "*") 
        string(APPEND exp "${i}" "=") 
        string(APPEND exp "${value}" " ") 
    endforeach() 
    message(${exp})
endforeach()

cmake基础语法

cmake基础语法

  • cmake中的命令：COMMAND()

    • CMake 构建过程的操作基本使用各种的命令完成
    • 示例：project(test),add_executable(hello hello.cpp)

  • 条件判断 if()/endif()

    • ~~~
      if()

      elseif() # optional block, can be repeated else() # optional block endif()

    • items 是一个 list，里面的元素用分号 ; 或者空格分割开来

    • 例如：foreach(index A B C D)或foreach(index RANGE 9)

    设置变量 set()/unset()

    • CMake 的变量可以分为 普通变量、系统环境变量。
    • set() 命令可以设置普通变量、系统环境变量、和缓存。
    • 设置普通的变量，将 value 的值赋值给 variable，后面的参数是可选的，代表有效域:set(<variable> <value>... [PARENT_SCOPE])
    • 设置系统环境变量:set(<variable> <value>... CACHE <type> <docstring> [FORCE])
    • 访问变量${variable}
