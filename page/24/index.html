Linux find命令

Linux find命令

  • ​find命令可以搜索指定目录下的文件。

  • 语法：find path -option [expression]

  • 其中path表示要搜索的目录，option表示查找参数

  • 实例：

    • find //查找当前目录下的所有文件
find ./ -name "a.txt" //查找当前目录下名称为a.txt的文件
find ./ -maxdepth 2 -name "a.txt"//限制深度为2查找

Linux grep命令

Linux grep命令

  • 当前目录，搜索preamp字段的所有文件，并打印出所在行：

    • grep preamp *

output:
preamp.cpp:double preamp(double *x,double *par)
preamp.cpp:void preamp()
preamp.cpp:  TF1 *fp=new TF1("fp",preamp,0,200,4);
preamp.cpp~:double preamp(double *x,double *par)
preamp.cpp~:void preamp()
preamp.cpp~:  TF1 *fp=new TF1("fp",preamp,0,200,4);

详细使用方法

  • 语法：grep [-abcEFGhHilLnqrsvVwxy][-A<显示列数>][-B<显示列数>][-C<显示列数>][-d<进行动作>][-e<范本样式>][-f<范本文件>][--help][范本样式][文件或目录...]

  • 参数说明：

    • -a 或 --text : 不要忽略二进制的数据。
-A<显示行数> 或 --after-context=<显示行数> : 除了显示符合范本样式的那一列之外，并显示该行之后的内容。
-b 或 --byte-offset : 在显示符合样式的那一行之前，标示出该行第一个字符的编号。
-B<显示行数> 或 --before-context=<显示行数> : 除了显示符合样式的那一行之外，并显示该行之前的内容。
-c 或 --count : 计算符合样式的列数。
-C<显示行数> 或 --context=<显示行数>或-<显示行数> : 除了显示符合样式的那一行之外，并显示该行之前后的内容。
-d <动作> 或 --directories=<动作> : 当指定要查找的是目录而非文件时，必须使用这项参数，否则grep指令将回报信息并停止动作。
-e<范本样式> 或 --regexp=<范本样式> : 指定字符串做为查找文件内容的样式。
-E 或 --extended-regexp : 将样式为延伸的普通表示法来使用。
-f<规则文件> 或 --file=<规则文件> : 指定规则文件，其内容含有一个或多个规则样式，让grep查找符合规则条件的文件内容，格式为每行一个规则样式。
-F 或 --fixed-regexp : 将样式视为固定字符串的列表。
-G 或 --basic-regexp : 将样式视为普通的表示法来使用。
-h 或 --no-filename : 在显示符合样式的那一行之前，不标示该行所属的文件名称。
-H 或 --with-filename : 在显示符合样式的那一行之前，表示该行所属的文件名称。
-i 或 --ignore-case : 忽略字符大小写的差别。
-l 或 --file-with-matches : 列出文件内容符合指定的样式的文件名称。
-L 或 --files-without-match : 列出文件内容不符合指定的样式的文件名称。
-n 或 --line-number : 在显示符合样式的那一行之前，标示出该行的列数编号。
-o 或 --only-matching : 只显示匹配PATTERN 部分。
-q 或 --quiet或--silent : 不显示任何信息。
-r 或 --recursive : 此参数的效果和指定"-d recurse"参数相同。
-s 或 --no-messages : 不显示错误信息。
-v 或 --revert-match : 显示不包含匹配文本的所有行。
-V 或 --version : 显示版本信息。
-w 或 --word-regexp : 只显示全字符合的列。
-x --line-regexp : 只显示全列符合的列。
-y : 此参数的效果和指定"-i"参数相同。

Python gui库Tkinter

Python 库Tkinter​

  • Python库Tkinter(Tk接口)是一个标准的GUI工具包接口，可以运行在大多数平台上。

  • 创建简单gui窗口：

    • ~~~
      from tkinter import *
      window = Tk()
      window.title(“Welcome to LikeGeeks app”)
      window.mainloop()

  • 设置标签字体大小和窗口大小：

    • ~~~
      lbl = Label(window, text=”Hello”, font=(“Arial Bold”, 50))//font size
      window.geometry(‘350×200’)//setting window size

  • 示例：

    • from tkinter import *

window = Tk()
window.title("Welcome to LikeGeeks app")
window.geometry('350x200')
lbl = Label(window, text="Hello")
lbl.grid(column=0, row=0)
btn = Button(window, text="Click Me")
btn.grid(column=0, row=0)
window.mainloop()

    • 显示效果：

    • 按钮的显示位置设置在第二行，相应在窗口的位置设为1，如果设置为0将会覆盖文本标签。

  • 详细教程：https://likegeeks.com/python-gui-examples-tkinter-tutorial/

Latex使用BibTex添加参考文献

Latex使用BibTex添加参考文献

  • 创建bib文件，命令生成空白文件，后缀名是.bib即可。

  • 打开bib文件，添加内容，示例：

    • ~~~
      @article{Liddick,
      author={S.N.Liddick et al.},
      journal = {PHYSICAL REVIEW LETTERS},
      volume = {97},
      number = {082501},
      year = {2006},
      }
    • 第一行指定生成的参考文献格式，第二行指定生成的bib文件名，不用加后缀。

  • 在论文主体中添加引用的格式如下：\cite{Liddick}

  • 编译的顺序按照如下进行：

    • 用Latex编译文档，生成引用
    • 用BibTex编译bib文档
    • Latex编译文档，将引用参考文献添加至文体
    • 通过dviPDF查看PDF

参考文献可选格式

  • plain，按字母的顺序排列，比较次序为作者、年度和标题；
  • unsrt，样式同plain，只是按照引用的先后排序；
  • alpha，用作者名首字母+年份后两位作标号，以字母顺序排序；
  • abbrv，类似plain，将月份全拼改为缩写，更显紧凑；
  • ieeetr，国际电气电子工程师协会期刊样式；
  • acm，美国计算机学会期刊样式；
  • siam，美国工业和应用数学学会期刊样式；
  • apalike，美国心理学学会期刊样式；

编辑bib文件

  • 编辑bib文件可以直接用文本编辑器编辑，并添加内容，如记事本、word等。
  • 也可用WinEdt编辑，可以直接拖动bib文件至软件窗口，等文件打开后即可编辑。

centos7上安装apache服务器

  • Apache http服务器是一款免费开源跨平台的http服务器，提供的功能强大，且易于扩展。

  • centos7下安装apache服务器执行以下命令：

    • ~~~
      sudo yum install httpd

  • 查看版本：

    • ~~~
      httpd -v
      output：
      Server version: Apache/2.4.35 (IUS)
      Server built: Dec 8 2018 02:08:22
    • 新版本如果用service启动的话，会提示用systemctl启动。
