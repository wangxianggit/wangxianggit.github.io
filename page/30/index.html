cmd新建文件

  • ​ 建立空文件方法
    • cd.>a.txt, cd. 相当于空操作重定向到文本文件，即建立空文件
    • copy nul a.txt, nul表示空设备，它不可见存在于每个目录中，可以看成是一个特殊的文件；一般可以把输出写入到nul，来达到屏蔽输出的目的。在此表示将空设备复制到文本文件中，即创建了一个空文件。
    • type nul>a.txt, 表示显示空设备的内容，并重定向到文本文件中。
    • echo a 2>a.txt, “2”表示错误输出的句柄，在此没有错误输出即创建空文件。
    • 即只要没输出并重定向到文件中就可以了。
  • 建立非空文件的方法
    • 同上面类似，只要将非空内容重定向到文件即可。
    • 常用命令是echo，或type或copy等

Python特殊变量__name__

  • Python脚本中常常有这样的语句：if __name__ == ‘__main__‘，用于执行某个脚本，但是这里涉及到一个python特殊变量 __name__ ，这里是在变量前后都加上了双下划线，所以是python定义的特殊变量。

  • 关于python需要解释以下几点：

    • Python的模块可以独立运行，同时模块之间也可以通过import互相引用
    • import的操作会执行所import的模块
    • __name__ 特殊变量就是标识模块的名字的一个系统变量，若当前模块是主模块，模块名字就是 __main__ ，如果模块是被导入， __name__的值为模块名字
    • 通过模块名称的判断可以确定当前模块是否是主函数，并执行相应操作

  • 示例代码

    • ~~~
      #Py1.py
      #!/usr/bin/env python
      def test():
      print ‘name = ‘,name
      if name == ‘main‘:
      test()

    • 执行结果：

    • __name__=__main__
Py1.py __name__=Py1

  • 通过结果可以知道，Py2.py直接执行，那么内建变量__name__的值为__main__，否则为模块的名字，通过这个特性可以在if语句里面添加测试代码，减少BUG，提高程序的可靠性。

    Python3标准数据类型bytes

    • bytes类定义

      • ~~~
        “””

        bytes(iterable_of_ints) -> bytes
bytes(string, encoding[, errors]) -> bytes
bytes(bytes_or_buffer) -> immutable copy of bytes_or_buffer
bytes(int) -> bytes object of size given by the parameter initialized with null bytes
bytes() -> empty bytes object

Construct an immutable array of bytes from:
  - an iterable yielding integers in range(256)
  - a text string encoded using the specified encoding
  - any object implementing the buffer API.
  - an integer
"""

    • 升级到Python3后，字符串和bytes类型分开，字符串是以字符为单位进行处理的，bytes类型是以字节为单位处理。

    • bytes数据类型在所有的操作和使用以及内置方法上和字符串类型基本一样，序列一旦创建不可改变。

    • bytes对象只负责以二进制字节序列的形式记录所需要记录的对象，至于该对象到底表示什么(比如到底是什么字符)则由相应的编码格式解码决定。

    • Python3中，bytes对象通常用于网络数据传输、二进制图片和文件保存等等，可以通过调用bytes()生成bytes实例。

    • bytes实例值的形式为b’xxxx’，其中’xxxx’是一至多个转义的十六进制字符串(例如\x12)组成的序列，每个十六进制代表一个字节。

    • 对同一个字符串采用不同的编码方式生成的bytes对象，会生成不同的值。

    • Python3最重要的新特性大概要算是对文本和二进制数据作了更为清晰的区分。文本总是Unicode，由str类型表示，二进制数据则由bytes类型表示。

    • 示例代码

      • a=b'jdk'
a_b=a.decode('utf-8')
print(a_b)
type(a_b)
