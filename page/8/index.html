windows下安装tensorflow

  • 使用pip安装tensorflow：

    • cpu版本：pip install --ignore-installed --upgrade tensorflow
    • gpu版本：pip install --ignore-installed --upgrade tensorflow-gpu
    • 使用国内镜像：pip install tensorflow==2.0.0 -i https://pypi.tuna.tsinghua.edu.cn/simple

  • 测试代码：

    • #TensorFlow使用图(Graph)来表示计算任务；并使用会话(Session)来执行图，通过Session.close()来关闭会话（这是一种显式关闭会话的方式）。会话方式有显式和隐式会话之分。
import tensorflow as tf
hello = tf.constant('Hello, TensorFlow!')  #初始化一个TensorFlow的常量
sess = tf.Session()  #启动一个会话
print(sess.run(hello))

Python pdb调试命令

  • ENTER (重复上次命令)

  • c (继续)直到遇到下一个断点

  • l (查找当前位于哪里)

  • s (进入子程序,如果当前有一个函数调用，那么 s 会进入被调用的函数体)

  • n(ext) 让程序运行下一行，如果当前语句有一个函数调用，用 n 是不会进入被调用的函数体中的

  • r (运行直到子程序结束)

  • !<python 命令>执行shell命令

  • h (帮助)

  • a(rgs) 打印当前函数的参数

  • j(ump) 让程序跳转到指定的行数

  • l(ist) 可以列出当前将要运行的代码块

  • p(rint) 最有用的命令之一，打印某个变量

  • q(uit) 退出调试

  • r(eturn) 继续执行，直到函数体返回

  • 使用ipdb可以显示彩色环境，安装如下：

    • pip install ipdb -i https://pypi.tuna.tsinghua.edu.cn/simple
