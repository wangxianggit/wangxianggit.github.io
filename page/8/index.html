Opencv处理三通道图像

  • 单通道图，即灰度图，每个像素点只有一个值代表颜色，像素值在0到255之间，0是黑色，255是白色，中间值是一些不同等级的灰色。3通道灰度图其实只有一个通道有值，其他两个通道的值是零。

  • 一般每个像素点都有三个值，即三通道图。例如RGB图片即为三通道图片，RGB色彩模式是工业界的一种颜色标准，是通过对红(R)、绿(G)、蓝(B)三个颜色通道的变化以及它们相互之间的叠加来得到各式各样的颜色的，RGB即是代表红、绿、蓝三个通道的颜色，这个标准几乎包括了人类视力所能感知的所有颜色，是目前运用最广的颜色系统之一。

  • 使用opencv库可以对图像的三通道分别进行处理，处理中需要用到cv2模块，在anaconda环境中安装：

    • conda install --channel https://conda.anaconda.org/menpo opencv

Winedt注册表破解

Winedt10注册表破解

  • 在windet每次退出时运行一段代码，修改注册表信息，使试用天数一直设为1，可以一直免费试用。

  • 打开winedt软件：

    • ~~~
      option 》option interface》Advanced Configuration》Event Handlers》Exit

  • 单击保存，即当软件自动退出时执行该语句，将试用天数设为1。

  • 每次运行可以检验相关注册表项对应记录的变化：

    • 首先桌面开始搜索窗口中输入“regedit”,回车打开windows系统的注册表。
    • 在注册表中找到“HKEY_CURRENT_USER–>Software–>WinEdt 10”这一项，可以看到左侧窗口有“Inst”这一项。这就是记录试用期过了多少天的注册表项。
    • 现在回到WinEdt程序，在Exit文件上右击，选择Execute Script，或者关闭WinEdt程序。
    • 再回到注册表编辑器，按“F5”刷新注册表显示，可以看到“Inst”项消失，说明相应记录改变。

latex论文用图修改前缀

    • \usepackage[footnotesize]{caption2}
