Centos7 error：no wifi-adapter found

  • 新安装的centos7和window10双系统，开机打开无线网络出现错误：

    • no wifi-adapter found

  • 解决方法,管理员权限下输入以下命令：

    • rfkill unblock all

ssh启动错误：no hostkeys available— exiting

  • ssh服务启动不了，查看出现错误原因：

    • /usr/sbin/sshd -T
    • 若输出错误存在问题：no hostkeys available— exiting，则错误原因是ssh密钥出现问题。

  • root权限下，重新生成密钥：

    • ssh-keygen -t dsa -f /etc/ssh/ssh_host_dsa_key
    • ssh-keygen -t rsa -f /etc/ssh/ssh_host_rsa_key

  • 修改密钥权限：

    • chmod 600 /etc/ssh/ssh_host_dsa_key
    • chmod 600 /etc/ssh/ssh_host_rsa_key

  • 重启ssh服务即可。

Centos7 ssh常用命令

Centos7 ssh常用命令

  • 安装程序包：yum install -y openssh-server

  • 启动ssh服务：

    • 启动：systemctl start sshd

    • 重启：systemctl restart sshd

    • 开机启动: systemctl enable sshd

    • 开启防火墙:

    • firewall-cmd --add-service=ftp/tcp --permanent
firewall-cmd --add-service=http/tcp --permanent
firewall-cmd --reload

  • 启动ssh命令需要在root权限下运行。

cmake打印乘法表

cmake打印乘法表

  • CMakeLists.txt 文件代码：

    • cmake_minimum_required(VERSION 2.8.11) 
project(test) 
foreach(i RANGE 1 9) 
    set(exp "") 
    foreach(j RANGE 1 ${i}) 
        math(EXPR value "${j} * ${i}") 
        string(APPEND exp "${j}" "*") 
        string(APPEND exp "${i}" "=") 
        string(APPEND exp "${value}" " ") 
    endforeach() 
    message(${exp})
endforeach()
