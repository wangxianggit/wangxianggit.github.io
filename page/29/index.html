c++代码书写规范

C++代码书写规范

头文件保护

  • 使用#define进行头文件保护，为保证唯一性头文件保护的命名可以基于项目代码路径，例如对于头文件Main\Src\File.h，可以设置如下保护：

    • #ifndefine         MAIN_SRC_FILE_H

#define            MAIN_SRC_FILE_H

…

#endif

### 作用域

  • 尽量不要使用全局变量，全局变量起的作用往往仅是一个静态变量。

  • 如果非要使用全局变量，考虑使用单例模式代替。

  • 多线程中使用的全局变量，不要使用class类型（包括STL容器），避免class内部的实现非线程安全而导致bug。

类

  • 声明顺序按照以下规则：

    • 成员先后顺序：typedef、enum、常量、构造析构函数、成员函数，成员变量。

    • 域先后顺序：public、protected、private。

    • 例：

    • ~~~
      Class MyClass
      {
      public:

           voidFoo();

      protected:

           voidBar();

      private:

           voidFunc();

      }

  • 只有数据时，使用结构体，否则使用类。

  • 尽量使用组合代替继承。

    • 如果类中有一个virtual函数，则析构函数必须声明为virtual。
    • 所有继承必须为public，私有继承使用其它实现方式替代。

  • 尽量避免使用多重继承，除非是COM。

  • 尽量使用const修饰变量。

注释的使用

  • 头文件的注释格式如下：

    • ~~~
      /****

      • @description:

      • @author:      name of author

      • @date:        2018.11.10

      • @last modified author:

      • @last modified date:

      • @ copyright “the company” 2018 -

      • ****/

  • 函数注释，重要或难理解的函数，需要在函数声明前加上注释。

  • 对于很难理解、重要、实现精彩的部分必须加上注释。

  • 对于那些来不及实现、暂未实现、需要去实现的地方加上TODO注释。

  • ~~~
    // TODO: Fly to the moon.

  • 变量前缀为小写，根据变量类型加如short/int/size_t..，加n，如int nVal =1;

  • 使用空格代替table，在VS里将table设置成4个空格，每个级别进行4个空格的缩进。

  • 花括号另起一行使用。

  • 条件、循环、开关等语句，关键字后加空格再使用，例如If ();

其他

  • new/delete等堆操作，必须成对出现，尽量遵从谁分配谁释放的原则。
  • AddRef()/Release()必须成对出现，尽量遵从谁分配谁释放的原则。
  • 指针、句柄使用完释放后必须设置成NULL。
  • 函数功能要单一，避免耦合多功能，严禁CreateAndNavigate这种函数。
  • 指针的空值比较使用NULL，而不是0。
  • 浮点数的零值比较严禁使用== 0。

Altium焊盘pad和过孔via区别

Altium焊盘和过孔

  • 图形外观上左边是焊盘Pad，右边是过孔Via
  • 焊盘也称为插件孔，用于放置与连接插件；过孔用于上下两层之间的连接，常用在多层Multi-Layer。
  • Pad就是PCB上面可以使用焊锡的那些裸露出来的铜面。它有不穿孔和穿孔两类，不穿孔的，就是我们常用的贴封装器件的引脚放置的地方，就是不穿孔的Pad；穿孔的，上面可以使用焊锡焊接插针，比如我们常用的插针的孔，就是穿孔的Pad，上图中左边的就是穿孔的Pad。
  • Pad可以设置大小、形状有Top Paste和Bottom Paste，可以涂焊锡。

Altium_designer中三种不同覆铜方式

Altium 不同覆铜方式

  • 三种覆铜方式：Fill,Polygon pour和Solid region，三种不同的覆铜方式分别完成不同的功能。
  • Fill即在一个区域内直接用铜皮覆盖，且不区分是否存在焊盘或过孔。
    • 因此在Fill填充的区域如果存在网络，线路或过孔等都将连接在一起，不正确的使用容易造成短路。
    • Fill填充的区域面积一般比较小，同时Fill的区域只能是矩形不可随意绘制图形。
    • Fill在布线前完成。
  • Polygon pour即铺铜也称为灌铜，作用和Fill相近但有以下区分
    • 灌铜可以主动区分过孔和网络，如果过孔与焊点不属于同一个网络，灌铜将根据设置好的规则将它们分开并保持安全距离。
    • 灌铜的目的是提高电路的抗干扰能力、减小辐射干扰、高频PCB电路板常常选用。
    • 灌铜功能可以自动删除死铜，在铺铜时，点击Tools-Preferences，在PCB Editor 栏的Interactive Routing 处选Push Obstacle （遇到不同网络的走线时推挤其它的走线，Ignore Obstacle为穿过）模式并选中Automatically Remove Loops（自动删除多余的走线）。
  • Solid region即在在覆铜区域内整片覆盖成一个整体铜膜。
    • 覆铜区域内不允许存在其它网络的焊盘 过孔 铜膜走线，否则会造成短路。
    • 实心覆铜可以绘制任意形状，也可以由其它图形转换而来，为了能绘出任意图形的实心覆铜，可以先在PCB板的顶部丝印层top overly，用划线工具画出任意形状的图形，随后框选它，然后点击菜单tools—-convert—-create region from selected primitives—-创建实心覆铜区域。
    • 实心覆铜常用于创建任意形状的焊盘、铜膜走线、插接件、连接件、在制作PCB封装时经常使用。

Altium Designer二维三维视图切换

AD视图切换​

  • 由菜单栏切换到3D视图，依次点击菜单：
    • 视图->3D布局模式
    • 回到菜单栏点击视图->2D布局模式，切换回3D视图
  • 由快捷键切换：按键盘3到3D视图模式，在3D视图模式下按2回到2D视图模式
  • 在3D视图模式下可以进行的操作：
    • 缩放：Ctrl+右键拖动鼠标，或Ctrl+滚动鼠标滚轮或PgUp/PgDn键
    • 平移：右键拖动鼠标，或标准windows鼠标滚轮控件
    • 旋转：Shift+右键拖动鼠标。
