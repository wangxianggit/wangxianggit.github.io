Latex使用BibTex添加参考文献

Latex使用BibTex添加参考文献

  • 创建bib文件，命令生成空白文件，后缀名是.bib即可。

  • 打开bib文件，添加内容，示例：

    • ~~~
      @article{Liddick,
      author={S.N.Liddick et al.},
      journal = {PHYSICAL REVIEW LETTERS},
      volume = {97},
      number = {082501},
      year = {2006},
      }
    • 第一行指定生成的参考文献格式，第二行指定生成的bib文件名，不用加后缀。

  • 在论文主体中添加引用的格式如下：\cite{Liddick}

  • 编译的顺序按照如下进行：

    • 用Latex编译文档，生成引用
    • 用BibTex编译bib文档
    • Latex编译文档，将引用参考文献添加至文体
    • 通过dviPDF查看PDF

参考文献可选格式

  • plain，按字母的顺序排列，比较次序为作者、年度和标题；
  • unsrt，样式同plain，只是按照引用的先后排序；
  • alpha，用作者名首字母+年份后两位作标号，以字母顺序排序；
  • abbrv，类似plain，将月份全拼改为缩写，更显紧凑；
  • ieeetr，国际电气电子工程师协会期刊样式；
  • acm，美国计算机学会期刊样式；
  • siam，美国工业和应用数学学会期刊样式；
  • apalike，美国心理学学会期刊样式；

编辑bib文件

  • 编辑bib文件可以直接用文本编辑器编辑，并添加内容，如记事本、word等。
  • 也可用WinEdt编辑，可以直接拖动bib文件至软件窗口，等文件打开后即可编辑。

centos7上安装apache服务器

  • Apache http服务器是一款免费开源跨平台的http服务器，提供的功能强大，且易于扩展。

  • centos7下安装apache服务器执行以下命令：

    • ~~~
      sudo yum install httpd

  • 查看版本：

    • ~~~
      httpd -v
      output：
      Server version: Apache/2.4.35 (IUS)
      Server built: Dec 8 2018 02:08:22
    • 新版本如果用service启动的话，会提示用systemctl启动。

win8.1下anaconda环境变量配置

anaconda环境变量配置

  • windows下安装好anaconda后默认没有把环境变量添加到系统路径中去，所以在cmd中无法识别相应的命令，如pip，conda等。

  • 若安装anaconda的路径在D:\anaconda\

  • 在我的电脑->属性->高级系统设置->环境变量->PATH中加入一下路径：

    • D:\anaconda\ 
D:\anaconda\Scripts\ ##pip conda etc
D:\anaconda\Library\bin\

anaconda管理python版本

anaconda管理python2和python3

  • 使用命令conda info –envs查看已安装的python版本

    • ~~~
      C:\Users\guest>conda info –envs

      conda environments:

      base * D:\Anacondainstall
      py2 D:\Anacondainstall\envs\py2

  • 切换到python2命令：activate py2

  • 命令deactivate则推出python2环境。
