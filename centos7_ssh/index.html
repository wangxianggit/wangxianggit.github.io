Centos7 ssh常用命令

  • 安装程序包：yum install -y openssh-server

  • 启动ssh服务：

    • 启动：systemctl start sshd

    • 重启：systemctl restart sshd

    • 开机启动: systemctl enable sshd

    • 开启防火墙:

    • firewall-cmd --add-service=ftp/tcp --permanent
firewall-cmd --add-service=http/tcp --permanent
firewall-cmd --reload

  • 启动ssh命令需要在root权限下运行。
