Centos7 mongodb集群搭建

一、环境准备

  • 1、Centos7.9
  • 2、mongodb安装包，下载地址：https://www.mongodb.com/download-center/community
  • 3、三台机器IP分别是：10.10.0.14、10.10.0.118、10.10.0.119

二、mongdb数据库的安装 ,如下操作是分别在三台机器进行

  • 使用tar命令解压安装包，然后移动到/opt/mongodb目录下
  • 进入mongodb目录后创建db目录和日志文件：
  • 创建mongodb.conf文件：vi mongodb.conf
  • 配置开机启动：
    • cd /lib/systemd/system
    • vi mongodb.service
  • 分别在三台机器启动mongodb：systemctl start mongodb.service

三、配置mongodb副本集

  • 下面对mongodb副本集进行初始化

    • 1、进入其中一台机器的mongo shell操作 ，同bind_ip 一样
  • /opt/mongodb/bin/mongo –host 10.10.0.14 –port 27017
  • 2、使用admin库：use admin
  • 3、mongo副本配置：config={_id:"mongoTest",members:[{_id:0,host:"10.10.0.14:27017"},{_id:1,host:"10.10.0.118:27017"},{_id:2,host:"10.10.0.119:27017"}]}，初始化：rs.initiate(config)
  • 如果运行出错并且错误码是code 74 ，请关掉防火墙再试
  • 4、查看详细信息：rs.status()
