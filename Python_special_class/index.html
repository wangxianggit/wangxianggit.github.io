Python定制类

article content

  • python中 __xxx__ 形式的变量或函数名在程序上有特殊用途。

  • __slot__用于指定类可以添加的属性。

  • __len__()用于返回类成员变量的长度。

  • __str__()自定义方式返回类对象的名称，例：

    • ~~~
      class Student(object):

       def __init__(self, name):
     self.name = name
 def __str__(self):
     return 'Student object (name: %s)' % self.name

      print(Student(‘Tom’))
      Student object (name: Tom)

    • 这里输入对象名称后python调用函数__repr__，输出实例名称。

  • 另外python中还有其他定制类如__iter__ , __getitem__等等。
