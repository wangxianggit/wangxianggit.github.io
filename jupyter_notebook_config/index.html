jupyter notebook服务器配置

基本使用方法

  • 参考链接：https://jupyter-notebook.readthedocs.io/en/stable/index.html
  • ​ jupyter服务安装：
    • python版本要求：python3.3以上或python2.7
    • 安装anaconda，jupyter notebook也同时安装
    • 或在安装完python后使用pip安装：pip install jupyter
  • 启动jupyter：命令行下使用命令jupyter notebook
  • 打开一个指定文件：jupyter notebook <filename>.ipynb
  • jupyter notebook帮助：jupyter notebook --help

服务器配置

  • 安装后默认启动ip：http://127.0.0.1:8888

  • 配置文件路径：

    • Windows: C:\Users\USERNAME\.jupyter\jupyter_notebook_config.py

    • OS X: /Users/USERNAME/.jupyter/jupyter_notebook_config.py

    • Linux: /home/USERNAME/.jupyter/jupyter_notebook_config.py

    • 若没有相应的配置文件目录，需要手动生成：

      • jupyter notebook --generate-config

  • 密码设置：

    • 在命令行下使用命令：jupyter notebook password

      • ~~~
        jupyter notebook password
        Enter password: **
        Verify password: **
        [NotebookPasswordApp] Wrote hashed password to /Users/you/.jupyter/jupyter_notebook_config.json

    • 在命令行下启动jupyter notebook即可密码登录，运行。
