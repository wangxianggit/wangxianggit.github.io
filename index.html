latex表格内换行

latex表格内换行

  • 使用宏包：\usepackage{makecell}
  • 环境：tabular
  • 命令格式：\makecell[居中情况]{第1行内容 \\ 第2行内容 \\ 第3行内容 …}
  • 参数：
    • [c]是水平居中，[l]水平左居中，[r]水平右居中；
    • *[c]水平 + 垂直居中，*[l]垂直居中 + 水平左居中，*[r]垂直居中 + 水平右居中。
  • makecell里文本内容想换行，就用\\ 分隔即可；
  • “tabular环境里”不能用”enumerate环境”，即”只能手动枚举”。

Jetson Xavier NX Jetpack安装

Jetson Xavier NX Jetpack安装

  • JetPack SDK包括最新的Jetson Linux驱动程序包（L4T），具有Linux操作系统以及CUDA-X加速库和API。
  • JetPack SDK还包括用于主机和开发人员套件的示例，文档和开发工具，并支持更高级别的SDK，例如用于流视频分析的DeepStream和用于机器人技术的Isaac。
  • Jetson开发板安装JetPack 的方法总共有两种：1、SD卡镜像安装。2、使用NVIDIA SDK Manager安装，第一种更为简便。
  • 安装准备：
    • 1、格式化后的64G SD卡。
    • 2、Jetson Xavier NX Developer Kit SD卡镜像
    • 3、下载Etcher，并安装
  • 安装镜像：
    • 一、将Jetson Xavier NX Developer Kit镜像写入SD卡。
    • 1、打开balenaEtcher软件，单击“Flash from file”按钮，选择下载的zip文件。
    • 2、单击“Select target”按钮，选择对应的SD卡。
    • 3、单击“Flash”按钮，等待传输完成（大概10分钟左右）。
  • 首次启动及设置：
    • 1、将microSD卡（已写入JetPack映像）插入Jetson Xavier NX模块底部的插槽中。
    • 2、打开计算机显示屏的电源并连接它。
    • 3、连接USB键盘和鼠标。
    • 4、连接提供的电源。
    • 5、Jetson启动，跟随引导程序，一步步完成设置。
    • 6、登录后，完成安装。

NVIDIA Jetson AGX Xavier打开关闭风扇，解决风扇突然不转问题

  • 经常在Jetson Xavier刷完机开机后会出现风扇突然不转的问题，即使主板温度比较高(>40度)风扇依然不转。
  • 这个问题可以通过安装软件进行风扇开关的控制，以jetpack6.1版本为例。
    • 安装硬件温度检测工具sensors:sudo apt install lm-sensors
    • 安装成功后输入sensors，系统会显示当前的温度：
  • 开启全速模式，在终端输入：sudo nvpmodel --query,此时可以显示当前的工作模式。
  • 如果风扇没有开，则输入：jetson_clocks，此时风扇就会转动起来。
  • 修改风扇转速：在终端中输入cat /sys/devices/pwm-fan/target_pwm，此时将会显示风扇转速，如果为0则表示风扇不转，修改文件中的数字并重启可以控制风扇转速。
