torch_sparse系列安装方法

torch_sparse系列安装方法

  • 再安装torch_xxx系列包中，有的开源项目会给出他们使用的torch_scatter的pip install链接。
  • 但是安装后很可能出现和torch的cuda版本不匹配的问题，安装过程出现大量乱码并显示安装失败。
  • 安装完torch后，如果去https://pytorch-geometric.com/whl/torch-1.5.0.html下载相同版本的torch_sparse系列的whl文件并安装。然而，最后都会报错torch_sparse中有一串乱码无法识别，导致和torch对不上号。
  • 解决方法：
    • 成功安装torch后，使用指令pip install –no-index torch-cluster -f https://pytorch-geometric.com/whl/torch-1.x.0+cu10x.html（torch_cluster可直接修改为另外两样）进行安装即可。

Linux系统使用dmidecode查看系统的硬件信息

Linux系统使用dmidecode查看系统的硬件信息

  • dmidecode允许你在Linux系统下获取有关硬件方面的信息。dmidecode遵循SMBIOS/DMI标准，其输出的信息包括BIOS、系统、主板、处理器、内存、缓存等等。
  • DMI（Desktop Management Interface,DMI）就是帮助收集电脑系统信息的管理系统，DMI信息的收集必须在严格遵照SMBIOS规范的前提下进行。SMBIOS（System Management BIOS）是主板或系统制造者以标准格式显示产品管理信息所需遵循的统一规范。SMBIOS和DMI是由行业指导机构Desktop Management Task Force(DMTF)起草的开放性的技术标准，其中DMI设计适用于任何的平台和操作系统。
  • DMI充当了管理工具和系统层之间接口的角色。它建立了标准的可管理系统更加方便了电脑厂商和用户对系统的了解。DMI的主要组成部分是Management Information Format(MIF)数据库。这个数据库包括了所有有关电脑系统和配件的信息。通过DMI，用户可以获取序列号、电脑厂商、串口信息以及其它系统配件信息。
  • dmidecode的作用是将DMI数据库中的信息解码，以可读的文本方式显示。由于DMI信息可以人为修改，因此里面的信息不一定是系统准确的信息。
  • 使用方法：
  • 查看服务器的品牌与型号：dmidecode |grep -A 8 “System Information”
  • 查看CPU的信息：dmidecode|grep CPU
  • 查看处理器：dmidecode -t （processor, memory, bios…)

Linux性能监控命令glances

Linux性能监控命令glances

  • Glances 是一个由 Python 语言编写，使用 psutil 库从系统获取信息的跨平台命令行系统监视工具。
  • 通过 Glances，我们可以监视 CPU，平均负载，内存，网络流量，磁盘 I/O，其他处理器 和 文件系统 空间的利用情况。
  • Glances 是一个用来监视 GNU/Linux 和 FreeBSD 操作系统的 GPL 授权的自由软件，基于curses开发。

主要功能

  • CPU 信息 （用户的相关应用, 系统核心程序和空闲程序）
  • 总内存信息，包括了物理内存，交换空间和空闲内存等等
  • 之前的 1 分钟、5 分钟和 15 分钟平均的 CPU 负载
  • 网络链接的下行和上行速度
  • 处理器总数，以及其活动状态
  • 硬盘 I/O 相关（读写）速度详情
  • 当前挂载设备的磁盘使用情况
  • 高 CPU 和内存使用的进程名，和相关应用的位置
  • 在底部显示当前日期和时间
  • 将消耗最高系统资源的进程用红色标出

glances安装

  • 官方文档：https://glances.readthedocs.io/en/latest/install.html
  • glances安装源为epel，安装epel源后使用yum安装
  • 安装glances：yum -y install glances
  • 如果需要web页面监控，需安装web客户端工具： pip install bottle
  • 如果要输出温度，需要lm_sensors的支持: pip install pysensors, yum -y install lm_sensors
  • 自动安装脚本：curl -L https://bit.ly/glances | / bin / bash 或 wget -O- https://bit.ly/glances | / bin / bash

使用方法

  • 命令行直接使用glances命令，（Glances 的默认刷新频率是 1 （秒），但是你可以通过在终端指定参数来手动定义其刷新频率）。
  • 输出为html格式，配置web server输出：
    • # pip-python install jinja2
    • # glances -o HTML -f /var/www/html
    • 输出html格式时，需要jinja2的支持，需要先安装依赖。默认输出的文件名是glances.html，可以通过http://IP/glances.html访问。
  • 服务端按下面的命令启动glances
    • # glances -s -B 192.168.10.16
    • 客户端查看：# glances – c 192.168.10.16

latex表格内换行

latex表格内换行

  • 使用宏包：\usepackage{makecell}
  • 环境：tabular
  • 命令格式：\makecell[居中情况]{第1行内容 \\ 第2行内容 \\ 第3行内容 …}
  • 参数：
    • [c]是水平居中，[l]水平左居中，[r]水平右居中；
    • *[c]水平 + 垂直居中，*[l]垂直居中 + 水平左居中，*[r]垂直居中 + 水平右居中。
  • makecell里文本内容想换行，就用\\ 分隔即可；
  • “tabular环境里”不能用”enumerate环境”，即”只能手动枚举”。
