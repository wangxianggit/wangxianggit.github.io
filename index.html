Jetson Xavier NX常用组件安装

Jetson Xavier NX常用组件安装

  • 镜像安装版本为4.6.0
  • 驱动版本：head -n 1 /etc/nv_tegra_release
  • 内核版本：uname -r
  • 操作系统：lsb_release -i -r
  • cuda版本：nvcc -V
  • cuDNN版本：dpkg -l libcudnn8
  • opencv版本：dpkg -l libopencv
  • Tensorrt版本：dpkg -l tensorrt
  • NX开发套件中预装的python版本为2.7，但由于用到更多的版本为python3，
  • pip升级为最新版本：
  • 安装Jetson-stats管理工具：pip3 install jetson-stats，安装好后输入jtop可以查看各项资源使用情况。

NVIDIA Jetson AGX Xavier打开关闭风扇，解决风扇突然不转问题

  • 经常在Jetson Xavier刷完机开机后会出现风扇突然不转的问题，即使主板温度比较高(>40度)风扇依然不转。
  • 这个问题可以通过安装软件进行风扇开关的控制，以jetpack6.1版本为例。
    • 安装硬件温度检测工具sensors:sudo apt install lm-sensors
    • 安装成功后输入sensors，系统会显示当前的温度：
  • 开启全速模式，在终端输入：sudo nvpmodel --query,此时可以显示当前的工作模式。
  • 如果风扇没有开，则输入：jetson_clocks，此时风扇就会转动起来。
  • 修改风扇转速：在终端中输入cat /sys/devices/pwm-fan/target_pwm，此时将会显示风扇转速，如果为0则表示风扇不转，修改文件中的数字并重启可以控制风扇转速。

mediawiki上传office相关文件

mediawiki上传office相关文件

  • mediawiki默认情况下无法上传office相关文件,系统会报两种错误:
  • 1、The file is a corrupt or otherwise unreadable ZIP file. It cannot be properly checked for security.
  • 增加$wgAllowJavaUploads变量并设置为true
  • 2、File extension “.ppt” does not match the detected MIME type of the file (application/zip)
  • 打开includes/libs/mime/mime.types文件,找到application/zip行并在后面加上doc docx xls xlsx ppt pptx
