Linux性能监控命令glances

Linux性能监控命令glances

  • Glances 是一个由 Python 语言编写，使用 psutil 库从系统获取信息的跨平台命令行系统监视工具。
  • 通过 Glances，我们可以监视 CPU，平均负载，内存，网络流量，磁盘 I/O，其他处理器 和 文件系统 空间的利用情况。
  • Glances 是一个用来监视 GNU/Linux 和 FreeBSD 操作系统的 GPL 授权的自由软件，基于curses开发。

主要功能

  • CPU 信息 （用户的相关应用, 系统核心程序和空闲程序）
  • 总内存信息，包括了物理内存，交换空间和空闲内存等等
  • 之前的 1 分钟、5 分钟和 15 分钟平均的 CPU 负载
  • 网络链接的下行和上行速度
  • 处理器总数，以及其活动状态
  • 硬盘 I/O 相关（读写）速度详情
  • 当前挂载设备的磁盘使用情况
  • 高 CPU 和内存使用的进程名，和相关应用的位置
  • 在底部显示当前日期和时间
  • 将消耗最高系统资源的进程用红色标出

glances安装

  • 官方文档：https://glances.readthedocs.io/en/latest/install.html
  • glances安装源为epel，安装epel源后使用yum安装
  • 安装glances：yum -y install glances
  • 如果需要web页面监控，需安装web客户端工具： pip install bottle
  • 如果要输出温度，需要lm_sensors的支持: pip install pysensors, yum -y install lm_sensors
  • 自动安装脚本：curl -L https://bit.ly/glances | / bin / bash 或 wget -O- https://bit.ly/glances | / bin / bash

使用方法

  • 命令行直接使用glances命令，（Glances 的默认刷新频率是 1 （秒），但是你可以通过在终端指定参数来手动定义其刷新频率）。
  • 输出为html格式，配置web server输出：
    • # pip-python install jinja2
    • # glances -o HTML -f /var/www/html
    • 输出html格式时，需要jinja2的支持，需要先安装依赖。默认输出的文件名是glances.html，可以通过http://IP/glances.html访问。
  • 服务端按下面的命令启动glances
    • # glances -s -B 192.168.10.16
    • 客户端查看：# glances – c 192.168.10.16

latex表格内换行

latex表格内换行

  • 使用宏包：\usepackage{makecell}
  • 环境：tabular
  • 命令格式：\makecell[居中情况]{第1行内容 \\ 第2行内容 \\ 第3行内容 …}
  • 参数：
    • [c]是水平居中，[l]水平左居中，[r]水平右居中；
    • *[c]水平 + 垂直居中，*[l]垂直居中 + 水平左居中，*[r]垂直居中 + 水平右居中。
  • makecell里文本内容想换行，就用\\ 分隔即可；
  • “tabular环境里”不能用”enumerate环境”，即”只能手动枚举”。

Jetson Xavier NX Jetpack安装

Jetson Xavier NX Jetpack安装

  • JetPack SDK包括最新的Jetson Linux驱动程序包（L4T），具有Linux操作系统以及CUDA-X加速库和API。
  • JetPack SDK还包括用于主机和开发人员套件的示例，文档和开发工具，并支持更高级别的SDK，例如用于流视频分析的DeepStream和用于机器人技术的Isaac。
  • Jetson开发板安装JetPack 的方法总共有两种：1、SD卡镜像安装。2、使用NVIDIA SDK Manager安装，第一种更为简便。
  • 安装准备：
    • 1、格式化后的64G SD卡。
    • 2、Jetson Xavier NX Developer Kit SD卡镜像
    • 3、下载Etcher，并安装
  • 安装镜像：
    • 一、将Jetson Xavier NX Developer Kit镜像写入SD卡。
    • 1、打开balenaEtcher软件，单击“Flash from file”按钮，选择下载的zip文件。
    • 2、单击“Select target”按钮，选择对应的SD卡。
    • 3、单击“Flash”按钮，等待传输完成（大概10分钟左右）。
  • 首次启动及设置：
    • 1、将microSD卡（已写入JetPack映像）插入Jetson Xavier NX模块底部的插槽中。
    • 2、打开计算机显示屏的电源并连接它。
    • 3、连接USB键盘和鼠标。
    • 4、连接提供的电源。
    • 5、Jetson启动，跟随引导程序，一步步完成设置。
    • 6、登录后，完成安装。

Jetson Xavier NX常用组件安装

Jetson Xavier NX常用组件安装

  • 镜像安装版本为4.6.0
  • 驱动版本：head -n 1 /etc/nv_tegra_release
  • 内核版本：uname -r
  • 操作系统：lsb_release -i -r
  • cuda版本：nvcc -V
  • cuDNN版本：dpkg -l libcudnn8
  • opencv版本：dpkg -l libopencv
  • Tensorrt版本：dpkg -l tensorrt
  • NX开发套件中预装的python版本为2.7，但由于用到更多的版本为python3，
  • pip升级为最新版本：
  • 安装Jetson-stats管理工具：pip3 install jetson-stats，安装好后输入jtop可以查看各项资源使用情况。
