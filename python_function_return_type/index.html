python定义函数指定返回值类型

  • Python存在问题：
    • python是解释性语言，运行效率比较慢；
    • python是动态语言，后期维护的成本高，其中重要原因之一是没有类型检查，包括新建变量不需要声明以及指定类型等等。
  • python3.5之后，新增函数参数和返回值的类型指定和检查，新建变量时也可以指定类型。

基本类型指定

  • 例子中函数test指定输入参数类型，并指定返回值类型为str；

  • 在运行时如果输入和输出不是指定类型也不会报错，但是会出现警告，即本质是动态类型语言。
