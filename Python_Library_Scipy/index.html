Python库Scipy

Python库Scipy的使用

  • 官方教程链接：https://github.com/scipy-lectures/scipy-lecture-notes
  • 官方教程网页版：http://scipy-lectures.org/

Scipy模块

scipy.cluster 矢量量化/K均值算法
scipy.constants 物理和数学常数
scipy.fftpack 傅里叶变换
scipy.integrate 积分算法
scipy.interpolate 插值算法
scipy.io 数据输入和输出
scipy.linalg 线性代数算法
scipy.ndimage n维的图像包
scipy.odr ODR回归算法(极大似然估计)
scipy.optimize 寻找最佳值(拟合,矢量计算)
scipy.signal 信号过程
scipy.sparse 稀疏矩阵
scipy.spatial 空间数据结构算法
scipy.special 特殊数学函数
scipy.stats 统计算法

常用方法

  • Scipy模块都依赖于numpy模块，同时各自独立，导入scipy模块：

    • ~~~
      import numpy as np
      from scipy import stats #same for other sub-module

  • 其他的相关使用方法见官方教程。
