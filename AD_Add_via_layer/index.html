AD布线中添加过孔和板层

AD布线中添加过孔和板层

  • 在AD软件交互布线中可以添加过孔，过孔不能再产生冲突的位置添加。
  • 过孔属性设计规则在PCB Rules and Constraints Editors对话框中的Routing via Style添加。

添加过孔并切换板层

  • 在布线过程中按数字键盘的“*”或“+”键添加一个过孔并切换到下一个信号层。
  • 按“-”键添加一个过孔并切换到上一个信号层。

添加过孔不切换板层

  • 按“2”键添加一个过孔，但是仍然保持在当前布线层。
  • 单击确定过孔位置。

添加扇出过孔

  • 按数字键盘的“/”键为当前走线添加过孔。
  • 用这种方法添加过孔后将返回原交互式布线模式，可以立刻进行下一处网络布线。
  • 这个功能在需要放置大量过孔时可以节省时间。

布线中的板层切换

  • 在多层板上的焊盘或过孔布线时，可以通过快捷键L把当前线路切换到另一个信号层中。
  • 若需要切换板层需要先添加过孔才能实现。

PCB板的单层显示

  • 在PCB显示中按快捷键Shift+S可以实现单层显示。
  • 选择哪一层的标签，就显示哪一层。
  • 在单层显示中按快捷键Shift+S可以返回多层显示模式。
