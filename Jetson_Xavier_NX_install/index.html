Jetson Xavier NX常用组件安装

  • 镜像安装版本为4.6.0
  • 驱动版本：head -n 1 /etc/nv_tegra_release
  • 内核版本：uname -r
  • 操作系统：lsb_release -i -r
  • cuda版本：nvcc -V
  • cuDNN版本：dpkg -l libcudnn8
  • opencv版本：dpkg -l libopencv
  • Tensorrt版本：dpkg -l tensorrt
  • NX开发套件中预装的python版本为2.7，但由于用到更多的版本为python3，
  • pip升级为最新版本：
  • 安装Jetson-stats管理工具：pip3 install jetson-stats，安装好后输入jtop可以查看各项资源使用情况。
