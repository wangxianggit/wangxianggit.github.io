Python3语法变动

article content

  • ​ Print由语句变为函数

    • ~~~
      print ‘a’,’b’ -> print(‘a’,’b’)

  • 读取文件操作file()取消，统一使用open()；键盘读取函数raw_input(“message”)改为input(“message”)

  • 字符串编码是bytes类，bytes类解码为字符串，bytes序列一旦形成内容不可变

    • ~~~
      s=”abchd”
      b=s.encode(编码方式)//常用编码方式为”gbk”,”utf-8”,”ascii”等
      s1=b.decode(“编码方式”)

  • 除法运算

    • python 2.4.2以前

   10/3      结果为 3     

python 3.0

   10 / 3 结果为 3.3333333333333335

   10 // 3 结果为 3
