cmake基础语法

  • cmake中的命令：COMMAND()

    • CMake 构建过程的操作基本使用各种的命令完成
    • 示例：project(test),add_executable(hello hello.cpp)

  • 条件判断 if()/endif()

    • ~~~
      if()

      elseif() # optional block, can be repeated else() # optional block endif()

    • items 是一个 list，里面的元素用分号 ; 或者空格分割开来

    • 例如：foreach(index A B C D)或foreach(index RANGE 9)

    设置变量 set()/unset()

    • CMake 的变量可以分为 普通变量、系统环境变量。
    • set() 命令可以设置普通变量、系统环境变量、和缓存。
    • 设置普通的变量，将 value 的值赋值给 variable，后面的参数是可选的，代表有效域:set(<variable> <value>... [PARENT_SCOPE])
    • 设置系统环境变量:set(<variable> <value>... CACHE <type> <docstring> [FORCE])
    • 访问变量${variable}

### 字符串操作
    • Search and Replace 
string(FIND <string> <substring> <out-var> [...]) 
string(REPLACE <match-string> <replace-string> <out-var> <input>...) 

Regular Expressions 
string(REGEX MATCH <match-regex> <out-var> <input>...) 
string(REGEX MATCHALL <match-regex> <out-var> <input>...) 
string(REGEX REPLACE <match-regex> <replace-expr> <out-var> <input>...) 

Manipulation 
string(APPEND <string-var> [<input>...]) 
string(PREPEND <string-var> [<input>...]) 
string(CONCAT <out-var> [<input>...]) 
string(JOIN <glue> <out-var> [<input>...]) 
string(TOLOWER <string1> <out-var>) 
string(TOUPPER <string1> <out-var>) 
string(LENGTH <string> <out-var>) 
string(SUBSTRING <string> <begin> <length> <out-var>) 
string(STRIP <string> <out-var>) 
string(GENEX_STRIP <string> <out-var>) 

Comparison 
string(COMPARE <op> <string1> <string2> <out-var>) 

Hashing 
string(<HASH> <out-var> <input>) 

Generation

string(UUID <out-var> ...)

    终端打印信息

    • message([<mode>] "message to display" ...)
    • 支持的类型有：
      • none
      • STATUS
      • WARNING
      • AUTHOR_WARNING
      • SEND_ERROR
      • FATAL_ERROR
      • DEPRECATION

    数学运算

    • math(EXPR <variable> "<expression>" [OUTPUT_FORMAT <format>])
    • 计算数学表达是 experssion，然后将结果保存在 variable 中，后面是指定的格式为 10 进制还是 16 进制。
