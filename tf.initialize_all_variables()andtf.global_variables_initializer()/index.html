tf.initialize_all_variables()和tf.global_variables_initializer()

tf.initialize_all_variables()和tf.global_variables_initializer()

  • tf.global_variables_initializer() 是 tf.initialize_all_variables()的升级版

  • 变量初始化

变量初始化的标准形式：

init = tf.global_variables_initializer()

sess = tf.Session()

sess.run(init)

当然也可简写为：

tf.Session().run(tf.initialize_all_variables())
