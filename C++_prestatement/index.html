C++前置声明​

  • 使用前置声明可以解决在两个或多个头文件的互相包含问题。

  • 例如在头文件a.h和头文件b.h的内容分别如下：

    • ~~~
      //a.h
      #include “b.h”
      class A
      {
      ….
      private: 
      B b;
      };

  • 在这里加入前置声明不需要重新编译头文件a.h，但需要重新编译类B。

  • 使用前置声明也可以减少类A的大小，上面只是声明了类B的指针而不是整个头文件，因此代码量大大缩小。
